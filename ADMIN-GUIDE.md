# Zuo Group Website Editing Guide

This website is prepared for form-based editing through Decap CMS on Netlify.

Main editable files:

- `content/research.json`
- `content/publications.json`
- `content/people.json`
- `content/news.json`
- `content/photos.json`
- `content/contact.json`
- `content/facilities.json`

After the site is connected to Netlify, editors should use:

```text
/admin/
```

Recommended editing workflow:

1. Open the admin page.
2. Log in with an invited editor account.
3. Choose one content section.
4. Edit fields in the form.
5. Save the entry.
6. Wait for the site to redeploy.
